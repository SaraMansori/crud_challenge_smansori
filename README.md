## Stack used

- TypeScript
- React & React Hooks
- React Bootstrap
- Components
- Context API
- Jest & React Testing Library

## Starting the app & scripts

### Install dependencies

`npm install`

### Run mocked server

`npm run server`

### Run web server

`npm run start`

### Run tests

`npm test`

## Run format

`npm run format`
